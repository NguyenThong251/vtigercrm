## 4. 🔑 Các khái niệm cốt lõi khi phát triển Vtiger

### Event Handler (Sự kiện)

- `vtiger.entity.beforesave` — chạy trước khi lưu record
- `vtiger.entity.aftersave` — chạy sau khi lưu record
- Đăng ký qua `Vtiger_Event::register()`

### Webservice API Flow

```
1. GET /webservice.php?operation=getchallenge&username=<USER>
   → Nhận token

2. POST /webservice.php  (operation=login)
   → Nhận sessionName

3. Dùng sessionName cho các thao tác CRUD:
   - create / retrieve / update / delete / query / sync
```

### Module Structure (vtlib)

```
modules/
  <ModuleName>/
    <ModuleName>.php         ← CRMEntity class
    manifest.php             ← Module metadata
    models/
      Record.php
      Module.php
    views/
      List.php
      Detail.php
      Edit.php
    layouts/v7/modules/<ModuleName>/
      DetailViewBody.tpl
      EditViewBody.tpl
      resources/
        Detail.js
```

### Yêu cầu server Vtiger 8

- PHP: 8.1 hoặc 8.2
- MySQL: 5.7.23+ hoặc MariaDB 10.4+
- IonCube Loader (bắt buộc)
- RAM: tối thiểu 2GB (khuyến nghị 4GB+)
- Extensions PHP: `curl`, `gd`, `imap`, `mbstring`, `zip`, `openssl`, `xml`, `mysqli`, `pdo_mysql`
